# hello-cli-python

A simple Python script that prints a greeting message using a name passed from the command line.

## How to Run

1. Make sure you have **Python 3** installed.

2. Open a terminal (Command Prompt or PowerShell on Windows).

3. Go to the folder where the file is:

```sh
cd path/to/folder
```

4. Run the script:

```sh
python hello-cli-python.py YourName
```

Replace `YourName` with the name you want.

### Examples

```sh
python hello-cli-python.py Papadimitrioudev
```

Output:

```
Hello, Papadimitrioudev!
```

If you run without a name:

```sh
python hello-cli-python.py
```

It prints:

```
Hello, World!
```
