PapadimitriouDev Web Scraper

This is a Python script that downloads the homepage of:
https://papadimitrioudev.netlify.app
and prints all link texts and URLs using requests and BeautifulSoup.

Requirements:
- Python
- requests
- beautifulsoup4

Install libraries:
pip install requests beautifulsoup4

Run:
ScrapeLinksFromMySite.py
